Node. js es un entorno de ejecución de código abierto y multiplataforma que ejecuta código JavaScript fuera del navegador. Permite a los programadores crear aplicaciones de red potentes y escalables que son lo suficientemente ligeras para ejecutarse en un servidor o dentro de la nube

Proposito de Instalacion

Para Instalar NODE JS, deberas:

  1. Descargar NODE JS-> Descargar
  2. Abrir archivo de instalacion, seguir instrucciones (dar siguiente)
  3. Abrir NODE JS
  4. Personalizar con las extenciones nesesarias
  5. Listo, ahora a trabajar con NODE JS
